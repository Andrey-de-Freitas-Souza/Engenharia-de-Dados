Operações de movimentação em células e planilhas

Tecla de Atalho Ação ao Pressionar
→ Move o seletor uma célula para a direita.
← Move o seletor uma célula para a esquerda.
↑ Move o seletor uma célula para cima.
↓ Move o seletor uma célula para baixo.
Home Move o seletor para a primeira célula da linha atual.
End Ativa o modo Término e, com isso, você pode pressionar uma tecla de
direção para mover para a última célula da linha/coluna atual.
CTRL + → ou End + → Move o seletor para a última célula preenchida da linha atual.
CTRL + ← ou End + ← Move o seletor para a primeira célula preenchida da linha atual.
CTRL + ↑ ou End + ↑ Move o seletor para a última célula preenchida da coluna atual.
CTRL + ↓ ou End + ↓ Move o seletor para a primeira célula preenchida da coluna atual.
CTRL + Home Move o seletor para a célula A1.
CTRL + End Move o seletor para a última célula preenchida da planilha.
TAB Move o seletor para a direita e move para a próxima opção em uma caixa de diálogo.
SHIFT + TAB Move o seletor para a esquerda e move para a opção anterior em uma caixa de diálogo.
CTRL + TAB Alterna para a próxima guia em uma caixa de diálogo.
CTRL + SHIFT + TAB Alterna para a guia anterior em uma caixa de diálogo.
CTRL + PgDn Ativa a próxima planilha à direita.
CTRL + PgUp Ativa a próxima planilha à esquerda.

Operações de seleção em células e planilhas

Tecla de Atalho Ação ao Pressionar
CTRL + T Seleciona a região atual (se tiver dados). Pressionando CTRL + T novamente, seleciona a planilha inteira.
CTRL + SHIFT + O Seleciona todas as células com comentários.
F8 Ativa/desativa o modo de seleção estendida.
SHIFT + F8 Adiciona células não adjacentes à seleção, usando o mouse ou o teclado.
CTRL + SHIFT + End Estende a seleção das células até a última célula preenchida da planilha.
CTRL + SHIFT + Home Estende a seleção das células até a primeira célula da planilha.
CTRL + SHIFT + PgDn Seleciona a planilha atual e a próxima planilha.
CTRL + SHIFT + PgUp Seleciona a planilha atual e a anterior.
CTRL + Barra de Espaço Seleciona uma coluna inteira.
SHIFT + Barra de Espaço Seleciona uma linha inteira.
CTRL + SHIFT + * Seleciona a região em volta da célula ativa.

Primeira seção

Tecla de Atalho Ação ao Pressionar
F2 Edita a célula ativa e posiciona o cursor no final do conteúdo da célula.
CTRL + C Copia as células/objetos selecionados.
CTRL + X Recorta as células/objetos selecionados.
CTRL + V Cole o conteúdo (copiado ou recortado) na posição atual.
CTRL + ALT + V Exibe a caixa de diálogo Colar Especial. O comando Colar Especial fica disponível após copiar ou recortar.
CTRL + SHIFT + + Inserir células, linhas ou colunas.
CTRL + D Preenche para baixo com o conteúdo da célula ativa.
CTRL + R Preenche para o lado direito com o conteúdo da célula ativa.
CTRL + F Pesquisador usando o comando Localizar.
CTRL + U Substitui dados usando o comando Substituir.
CTRL + Y ou F4 Repete o último comando ou ação (se possível).
ALT + Enter Insere uma quebra de linha.
CTRL + Enter Insere um valor em modo de edição, insere o conteúdo da célula ativa em todas as células selecionadas.
SHIFT + Enter Confirma o valor inserido da célula e seleciona a célula acima.

Operações de Formatação

Tecla de Atalho Ação ao Pressionar
CTRL + N Aplica/remove o estilo negrito.
CTRL + I Aplica/remove o estilo itálico.
CTRL + U Aplica/remove o estilo sublinhado.
CTRL + SHIFT + & Aplica bordas externas à célula ou intervalo selecionado.
CTRL + SHIFT + ~ Aplica o formato de número geral.
CTRL + SHIFT + $ Aplica o formato de moeda com duas casas decimais.
CTRL + SHIFT + % Aplica o formato de porcentagem.
CTRL + SHIFT + ^ Aplica o formato de número científico.
CTRL + SHIFT + # Aplica o formato de data com dia, mês e ano.
CTRL + SHIFT + @ Aplica o formato de hora com hora e minuto.
CTRL + SHIFT + ! Aplica o formato de número com duas casas decimais e separador de milhar.
CTRL + 1 Abre a caixa de diálogo Formatar Células.

Operações de Inserção

Tecla de Atalho Ação ao Pressionar
CTRL + K Exibe a caixa de diálogo Inserir/Editar Hyperlink.
CTRL + ALT + T Criar tabela.
CTRL + SHIFT + ; Insere a data atual.
CTRL + SHIFT + : Insere a hora atual.
ALT + SHIFT + F1 Insere um gráfico em uma nova planilha de gráfico.
SHIFT + F2 Insere/edita um comentário na célula ativa.
